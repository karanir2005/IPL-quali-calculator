Waiting for data

Data is fetched from ESPN Cricinfo standings via a Node.js backend server running on port 5000. Click "Refresh live data" to fetch the latest standings. All data is fresh—no caching.

How the calculator decides

A team is marked as Qualified only when its current points are already above the highest possible final points of the fifth-place challenger. Tie-breakers and net run rate are not modeled, so equal-point situations are treated as not guaranteed.
